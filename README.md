# 🌟 Money Map Android Application

## Project: Money Map
- **Language**: Java
- **Theme**: Green Based 🌿

---

## Libraries
- 📊 mpandroidchart
- 🌈 Glide
- 📦 GlideCompiler
- 🖼️ ImagePicker

- 💵 Type: Income / Expenses

- 📂 Category: Smart tracking

### 2. Data Visualization
- 📈 Total Income
- 📉 Total Expenses

- 📊 Net Income

### 3. Data Saving
- 🔒 Secure SQLite Storage

### 4. Summary
- 📆 Monthly View
- 🗓️ Daily View
- 📊 Overall Analytics
- 📥 PDF Export

### 5. GUI
- 🎨 Colorful Buttons
- ✨ Smooth Transitions
- 🌟 Eye-catching Layouts
